# Лаба 2 — локальный чат (pipe + fifo)

Хост (процесс) форкается и создаёт клиентов двух типов соединений: неименованные каналы (`pipe`) и именованные (`mkfifo`). Хост принимает сообщения от клиентов и ретранслирует всем; сообщения выводятся в порядке отправки. Клиент, не отправляющий сообщений более минуты (и не являющийся хостом), должен быть завершён сигналом (по условию задачи).

### Сборка и запуск (Linux/POSIX, C++17)
- Сборка: `cmake -B build && cmake --build build && cp build/host . && rm -rf build`
- Запуск: `./host` — сам создаёт клиентов PIPE и FIFO.

### Про терминал (используем bash)
По умолчанию вызывается `/usr/bin/xterm -e "bash -c '<cmd>'"`. Можно заменить на любой эмулятор, который умеет `bash -c` (например, `gnome-terminal -- bash -c "<cmd>"`).

Рабочие файлы ввода/вывода чата лежат в `/tmp/lab2`: `host.in/out` и `<pid>.in/out` для клиентов.

### Сообщения
- Общее сообщение: просто текст — придёт всем.
- Личное: `@<pid> текст` — придёт только указанному PID и отправителю (pm). Неактивный клиент (>60 сек без отправки) завершается `SIGKILL`.
